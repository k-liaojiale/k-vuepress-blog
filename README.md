## 食用指南

1. 拷贝项目

```git
git clone https://github.com/k-liaojiale/k-vuepress-blog.git
```

2. 安装依赖
```sh
yarn add # 或者 npm i
```

3. 本地运行查看
```sh
yarn docs:dev # 或者 npm run docs:dev
```

4. 推送github
```sh
yarn deploy # 或者 npm run deploy
```
